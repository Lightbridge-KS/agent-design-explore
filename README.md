# Agent Design Explore

> Ideas, design/architectural patterns, and examples for building **agent-first
> systems & harnesses** — distilled from real-world OSS case studies.

A book built with [Astro](https://astro.build) + [Starlight](https://starlight.astro.build).

## Stack

- **Framework:** Astro + Starlight
- **Diagrams:** [`astro-mermaid`](https://github.com/astro-mermaid/astro-mermaid) (Mermaid code blocks in Markdown)
- **Deploy:** Netlify (static output, config in `netlify.toml`)
- **Package manager:** pnpm

## Content model

The book has two layers:

1. **Case Studies (OSS)** — `src/content/docs/oss/`
   The *raw input*: architecture walkthroughs of real open-source agent systems
   and harnesses.
2. **Concepts & Patterns** — _(added as the library grows)_
   *Synthesized* knowledge distilled from the case studies — reusable design
   patterns and architectural ideas.

## Adding a page

1. Drop a `.md` / `.mdx` file in the right folder under `src/content/docs/`
   (e.g. `oss/<slug>.md`). Use kebab-case filenames; add `title` + `description`
   frontmatter.
2. Wire it into the `sidebar` array in [`astro.config.mjs`](./astro.config.mjs)
   (sidebar order is manual, not autogenerated).
3. Mermaid diagrams render automatically from ```` ```mermaid ```` code blocks.

## Commands

All commands run from the project root:

| Command         | Action                                       |
| :-------------- | :------------------------------------------- |
| `pnpm install`  | Install dependencies                         |
| `pnpm dev`      | Start local dev server at `localhost:4321`   |
| `pnpm build`    | Build the production site to `./dist/`       |
| `pnpm preview`  | Preview the build locally before deploying   |
